Bindings

Per bindings si intendono gli strumenti che ci consentono di collegare e quindi rendere dinamici i template HTML con gli attributi metodi della classe (typescript) del componente

String interpolation

Con string interpolation visualizzo nel template il valore di una "variabile" typescript

ATTENZIONE: Le variabili che possono essere visualizzate sono solo ed esclusivamente attributi della classe.

In un progetto zone-less per garantire che la vista venga aggiornata se cambia il valore della variabile DEVO usare i signals e non variabili tradizionali.

Cosa sono i signals? Sono funzioni wrapper (funzioni contenitori, che avvolgono, gestiscono un valore che può essere di qualsiasi tipo: String, number, array, oggetti.
Un signal per la modifica dei valori mette a disposizione dei metodi, se vengono usati, il nuovo valore del signal verrà visualizzato nel browser

Quali sono gli step?

  1. Dichiarare un signal in typescript ed assegnarli un valore
  2. Inserire una chiamata alla funzione (signal è una funzione wrapper) fra doppie parentesi graffe.

Property Binding

Con property binding posso impostare il valore di un attributo di un tag html in modo dinamico usando variabili/signal typescript.

Per utilizzare property binding racchiudo il nome dell'attributo fra parentesi quadre, e fra le virgolette inserisco la "variabile/signal" che contiene il valore dell'attributo.

Attraverso style binding è possibile impostare uno o più stili di un elemento HTML in modo dinamico Si realizza utilizzando la sintassi [style.nomestiledaimpostare]="espressione typescript che determina il valore"
